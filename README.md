## В этом проекте обучалась модель управления транспортом на мультисенсорных данных и информации о локальных траекториях движения в условиях городского трафика.

## Обзор проекта 
Для обучения и тестирования модели использовался [CARLA Simulator](https://github.com/carla-simulator/carla/releases/tag/0.9.15). Одним из ключевых преимуществ этого симулятора является возможность синхронного подключения 
и настройки различных сенсоров, что делает его особенно подходящим для задач мультисенсорного восприятия. Распространяется как готовая предсобранная папка симулятора.<br>
После скачивания архив необходимо распаковать и запустить <ins>CarlaUE4.exe</ins>.<br>

Проект разделён на две основные части:

<ins>Часть 1 - Настройка окружения и сбор данных в симуляторе CarlaUE4:</ins><br>
  1. Подключение к симулятору Carla и настройка симуляции;<br>
  2. Генерация маршрута при помощи *GlobalRoutePlanner*;<br>
  3. Спавн транспортного средства в начальной точке маршрута и задание управляющего агента *BehaviorAgent*;<br>
  4. Настройка сенсоров(RGB-камеры, семантические камеры и лидары);<br>
  5. Задание функций обработки данных для каждого из сенсоров и формирование локальной траектории;<br>
  6. Запуск генерации данных.<br>

<ins>Часть 2 - Создание датасета и обучение модели управления</ins><br>
  1. Предобработка сгенерированных данных;<br>
  2. Разделение финального датасета на *train*, *val* и *test*;<br>
  3. Инициализация модели управления *CarlaControlModel*;<br>
  4. Запуск обучения;<br>
  5. Оценка модели на тестовой выборке;<br>
  6. Интерпретация результатов.
